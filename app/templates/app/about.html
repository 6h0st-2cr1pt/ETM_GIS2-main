About

Endemic Trees Monitoring System

A Geographic Information System for Monitoring Endemic Trees in the Philippines.

Overview

The Endemic Trees Monitoring System is a comprehensive GIS application designed to map, analyze, and visualize data about endemic tree species in the Philippines, with a focus on Negros Island. The system provides tools for data collection, visualization, and analysis to support conservation efforts and environmental research.

Features

  • Interactive GIS mapping
  • Multiple map layer support
  • Advanced analytics and visualizations
  • Data import via CSV or manual entry
  • Tree filtering by species and attributes
  • Complete dataset management
  • Customizable themes and settings
  • Historical population tracking
